# Installation

## Prérequis

- macOS 14+
- Xcode 15+
- Swift 5.7+

## Cloner le dépôt

```bash
git clone https://github.com/thierryH91200/PegaseUIData.git
cd PegaseUIData
open PegaseUIData.xcodeproj
```

Puis, compilez et exécutez l'application avec Xcode.
